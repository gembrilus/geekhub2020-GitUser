# geekhub2019
Git User
========

App for retrieve an info about any Githab-user

In app is used such libraries:
[DB Room][1],[Retrofit][2], [OkHttp][3], [Moshi][4], [Glide][5], [MVVM-pattern][6];
and tests: [Espresso][7], [Mockito][8], Hamcrest, JUnit and other
